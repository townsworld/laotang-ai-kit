# 背景音乐说明

## 音频文件位置
将神秘背景音乐文件放置在：`/public/sounds/mystical-ambient.mp3`

## 推荐音乐风格
- 环境音乐（Ambient）
- 神秘/冥想风格
- 舒缓、空灵的旋律
- 建议时长：3-5分钟循环播放

## 免费音乐资源推荐
1. **Pixabay Music**: https://pixabay.com/music/
   - 搜索关键词：mystical, ambient, meditation
   
2. **FreePD**: https://freepd.com/
   - 搜索：ambient, mysterious

3. **Incompetech**: https://incompetech.com/
   - 搜索：ambient, dark

## 示例音乐名称
- "Mystical Night"
- "Deep Meditation"
- "Spiritual Awakening"
- "Ancient Wisdom"

## 使用说明
1. 下载合适的免费音乐文件
2. 转换为 MP3 格式（如果需要）
3. 重命名为 `mystical-ambient.mp3`
4. 放入 `public/sounds/` 目录
5. 刷新页面即可使用

## 音量控制
默认音量设置为 30%（0.3），可在 `BackgroundMusic.tsx` 中调整。
