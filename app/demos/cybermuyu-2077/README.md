# 赛博电子木鱼 2077 - Cyber Wooden Fish 2077

一个融合赛博朋克风格与禅意体验的电子木鱼项目 🪵✨

## 项目特色 | Features

### 🎨 视觉设计
- **真实木鱼造型**：红色半球形设计，还原传统木鱼外观
- **金色莲花装饰**：中心雕刻莲花图案，寓意功德圆满
- **赛博朋克风格**：青色霓虹边框与紫色光晕效果
- **动态背景**：透视网格地板与渐变光晕营造科技氛围

### 🔊 音效优化
- **真实木鱼音色**：使用 Web Audio API 合成
  - 基频 520Hz（木鱼主音）
  - 二次谐波增强（1040Hz）
  - 低频共鸣（280Hz）
  - 低通滤波器模拟木质音色
- **短促敲击包络**：还原真实木鱼的"咚"声特点

### ⚡ 交互体验
- **点击反馈**：木鱼缩放动画 + 音效
- **功德计数**：实时显示累计功德值
- **漂浮文字**：随机显示祝福语（"功德 +1"、"福报 +1"、"南无阿弥陀佛"等）
- **响应式设计**：支持鼠标和触摸操作

## 技术栈 | Tech Stack

- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **Framer Motion** - 动画库
- **Web Audio API** - 音效合成
- **Tailwind CSS** - 样式系统
- **Next.js 14** - 应用框架

## 功能说明 | How to Use

1. **点击木鱼**：鼠标点击或触摸屏幕中央的木鱼
2. **积累功德**：每次点击功德 +1
3. **观察动画**：欣赏漂浮文字和光效
4. **聆听音效**：体验真实的木鱼敲击声

## 优化内容 | Optimizations

相比原始版本，本版本进行了以下优化：

1. **木鱼外观**：从抽象的赛博图形改为真实的木鱼造型
2. **音效系统**：从单一振荡器升级为多层次谐波合成
3. **视觉效果**：增强了赛博朋克风格的视觉元素
4. **交互体验**：优化了动画和反馈效果
5. **代码结构**：整合到 Next.js 项目，使用现代化组件架构
