# Ep.01 - 赛博朋克个人全息名片

> **Demo 路径:** `app/demos/01-cyber-profile/page.tsx`

## 📝 Prompt（提示词）

```
# Task

请在项目中创建一个新的 Demo 页面，路径为 `app/demos/01-cyber-profile/page.tsx`。

这是一个**"赛博朋克风格的个人全息名片"**，我要用它来作为我自媒体账号的第一个展示案例。

# Visual Style (Cyberpunk Vibe)

- **Background:** 深邃的黑底 (Slate-950)，带有隐约的网格线 (Grid pattern) 或 扫描线背景。

- **Color Palette:** 霓虹青色 (Neon Cyan) 和 赛博紫 (Cyber Purple) 的撞色。

- **Effects:**
    1.  **Glitch Effect:** 名字需要有轻微的故障/抖动文字效果。
    2.  **Glassmorphism:** 卡片背景要是半透明磨砂玻璃感，边框要有发光效果 (Glow)。
    3.  **Badge:** 技能标签要用高亮胶囊样式。

# Content Requirements

- **Avatar:** 使用一个圆形的占位图，带有旋转的边框光环。
- **Name:** 程序员老唐AI
- **Title:** AI Efficiency Hacker | 职场反内卷专家
- **Description:** "致力于用 AI 提早两小时下班。正在构建老唐的 AI 装备库..."
- **Action Buttons:**
    1.  Primary: "Get My Prompts" (高亮，带有 hover 发光动画)
    2.  Secondary: "返回装备库" (返回首页)

# Technical Implementation

- 使用 React + Tailwind CSS 实现所有样式。
- 使用纯 CSS animation 实现入场浮动动画和各种特效。
- 复用项目中的 lucide-react 图标。

请直接生成该页面的完整代码。
```

## 🎨 效果特性

1. ✅ **动态网格背景** - 流动的赛博网格
2. ✅ **扫描线效果** - 上下移动的扫描线
3. ✅ **旋转光环** - 双层反向旋转的边框
4. ✅ **Glitch 文字** - 故障风格的名字特效
5. ✅ **玻璃态卡片** - 磨砂玻璃 + 发光边框
6. ✅ **浮动动画** - 卡片和元素的浮动效果
7. ✅ **打字机效果** - Status 文字的打字动画
8. ✅ **渐变按钮** - Hover 发光的主按钮
9. ✅ **技能胶囊** - 高亮的技能标签
10. ✅ **响应式设计** - 完美适配移动端

## 🛠️ 技术栈

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React Icons
- 纯 CSS Animations

## 💡 使用方法

1. 复制上面的 Prompt
2. 粘贴给 Cursor/Windsurf 中的 AI
3. 回车，等待生成
4. 访问 `/demos/01-cyber-profile` 查看效果

- 创意作品集首页
- 赛博朋克风格项目
